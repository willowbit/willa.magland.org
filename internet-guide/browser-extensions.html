Browser Extensions

Woo-hoo! Almost done!

Browser extensions are bits of software that you layer on top of your web browser to give it extra features. For Firefox and other Firefox-based browsers (like Librewolf and Waterfox), you can download them at https://addons.mozilla.org/. On Chrome and Chrome-based browsers, you can find them at the Chrome Web Store.

While the right extensions can make your web browsing more private, there are a lot of ugly ones out there. In general, it’s best to have only a few, vetted browser extensions to reduce risk and keep things snappy.

Adblocking 🚫

Note

Google has blocked UBlock Origin in Chrome, hence why I recommend using a Firefox based browser instead. To install it anyway, follow this guide.

Having a good adblocker installed is probably the biggest step you can take to improve your web browsing experience and privacy. Despite what Google tries to make you think, an adblocker is essential security software for the modern web. Not only does a good adblocker block ads, but hundreds of weird trackers that trace your activity. I recommend uBlock Origin, as it’s the current gold standard for browsing safely. It does all of the above, while also warning you about suspicious websites, offering protection from viruses and malware.

Link to website
uBlock Origin https://ublockorigin.com/
Tip

If you’re using the Librewolf browser, it’s installed by default, so you don’t have to do anything!

Password Managers 🔒

Most likely the largest vulnerability in your web browsing right now is your passwords. You’re probably using similar passwords for each of your logins, which leaves you open to being “hacked,” but it’s not your fault!

Passwords are bad user design by nature. Luckily, there’s a solution that makes signing into things super fast, and it only requires memorizing one password: a password manager. It can autofill your login details almost instantaneously, and it also handles generating new, secure passwords for your new logins. Hell, it can even handle pesky 2-factor authentication codes! It does this all while being extremely secure, private, and convenient.

That sounds risky…

You might be feeling wary about giving a website all of your login information. But this is the zinger: by design, you don’t need to trust your password manager. Password managers use a special encryption technology that allows only you to access your passwords. That “one password” I mentioned earlier is called the master password, and is used to decrypt your login information, so in reality, the password manager can’t see a thing.

This video explains it better than I can, but back to the guide!

Tip

As long as you pick a generally well-known and reputable one, it doesn’t really matter which password manager you choose. I use (and pay for) Bitwarden, and my Dad enjoys 1Password, but there are other great ones out there!

Important

Please, next to installing uBlock Origin, if there is one thing you take away from this guide, it should be signing up for a password manager! It will change your life, I guarentee it. You don’t deserve having to remember a million passwords!
